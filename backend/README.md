# Backend - AI Smart Event Platform

This is a minimal Flask scaffold for the backend. Files are placeholders to be implemented.

- Create a virtualenv and install dependencies:

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

- Run the app:

```powershell
python run.py
```

- Health check: GET http://localhost:5000/api/health
